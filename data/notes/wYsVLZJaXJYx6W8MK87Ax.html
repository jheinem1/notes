Node

Nodes are simple data structures that store the next node in a Linked List and some values. There are a variety of node types...

Minimalistic

class Node<T> {
    Item element;
    Node next;
}

This is generally considered to be ideal as generics are more flexible.

OOP

class Node {
    Node next;
}

class PersonNode extends Node {
    String name;
    int phone[30];
    String email;
}

Monolithic

class PersonNode {
    Node next;
    String name;
    int phone[30];
    String email;
}
