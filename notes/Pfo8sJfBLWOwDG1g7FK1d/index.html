Shell Sort

Unlike Selection Sort, Shell Sort has the possibility of Ω(n)\Omega(n)Ω(n) sorting in some cases. It can also have a better average sort time

Although this sort was widely used, this is by-and-large replaced with sorting.quick-sort (Private).

Steps

  1. Perform an Insertion Sort starting at the ends of the array, and gradually shrink the internal until it is 1 (i.e., converges to normal insertion sort)

Implementation

public static void sort(Comparable[] array) {
    int n = a.length;
    int h = 1;
    while (h < N / 3) {
        for (int i = h; i < n; i++)
            for (int j = i; j >= h && less(a[j],a[j - h]); j -= h)
                exch(a, j, j - h);
        h = h / 3;
    }
}
